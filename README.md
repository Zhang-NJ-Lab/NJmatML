# NJmatML
## pip install NJmatML

Automates data-driven machine learning for materials science via NJmatML (材料数据可视化、机器学习建模与预测)

模块：数据读取，可视化，热图，特征选择，特征生成，重要性排名，机器学习建模，准确率计算，交叉验证，符号回归等等。封装了简便函数用于机器学习建模。
